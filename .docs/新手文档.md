# 新手文档

## 环境准备

- Node.js >= 20.0.0
- pnpm 10.4.1（通过 `corepack enable` 或 `npm install -g pnpm@10.4.1`）

## 安装依赖

```bash
pnpm install
```

## 启动各端

```bash
# 管理后台
pnpm start:admin

# 商城前台
pnpm start:mall

# 移动端（Expo）
pnpm start:mobile
```

## 常用命令

```bash
# 代码格式化
pnpm format

# 代码检查
pnpm lint

# 类型检查
pnpm type-check

# 构建
pnpm build:admin
pnpm build:mall
pnpm build:mobile
```

## 提交规范

提交信息需符合 `.commitlintrc.json` 要求：

```bash
type(scope): subject
```

允许的 `type`：build、chore、ci、docs、feat、fix、perf、refactor、revert、style、test 等。

## 注意事项

- 所有内部包统一使用 `workspace:^1.0.0` 引用
- 新增公共包请放到 `apps/public/` 目录，并在 `apps/package.json` 中聚合
- 新增业务应用请放到 `apps/projects/` 目录
